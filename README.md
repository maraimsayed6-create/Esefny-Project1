# Esefny - AI First Aid Assistant

## Description
Esefny is an AI-powered first aid assistant developed using Flutter and Python. The application helps users identify injuries, receive first-aid guidance, and locate nearby hospitals.

## Live API
The mobile application communicates with a deployed backend API hosted on Hugging Face Spaces:

https://mariamsayed17-first-aid-api.hf.space

## Dataset
https://drive.google.com/file/d/1c3tLncRANgq23FDs4g_5SebHNKTxRGRo/view

## Technologies
- Flutter
- Python
- TensorFlow
- Hugging Face Spaces
- SQLite

## Features
- Injury image analysis
- First aid recommendations
- Hospital locator
- User authentication
- AI-powered assistance

## Mobile Application

The Android application package (APK) is included in this repository as:

**app-debug.apk**

You can download and install the APK file to test the mobile application directly on an Android device.
