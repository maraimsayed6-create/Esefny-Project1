# Esefny - AI First Aid Assistant

## Description
Esefny is an AI-powered first aid assistant developed using Flutter and Python. The application helps users identify injuries, receive first-aid guidance, and locate nearby hospitals.

## Live API
The mobile application communicates with a deployed backend API hosted on Hugging Face Spaces:

https://mariamsayed17-first-aid-api.hf.space
